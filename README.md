🎵 Soundwave — Spotify-Inspired Music Streaming UI
A fully browser-native music streaming interface built with HTML5, CSS3, JavaScript, and Bootstrap 5. No frameworks, no bundlers, no backend — just open the file and play.

📸 Preview

A dark, Spotify-like UI featuring a sidebar library, featured banner, artist cards grid, track list with live EQ animations, and a fixed player bar at the bottom.

🛠️ Tech Stack

Technology               Version                    Purpose
HTML5                      —                    Structure & Audio API
CSS3                       —                    Animations, layout, dark theme
JavaScript (ES6+)          —                    Playback logic, state management 
Bootstrap                 5.3.0                 Responsive grid utilities 
Google Fonts               —                    DM Sans typeface

🎧 Features
Playback

Play / Pause with animated album art (morphs square → circle → square while playing)
Previous / Next track navigation
Seek bar — click anywhere on the progress track to jump
Real-time timestamps — current time and total duration update live
Auto-advance — automatically plays the next track on song end

Queue Controls

Shuffle mode — randomizes the next track selection
Repeat mode — loops the current track via audio.loop
Volume slider — smooth 0–100 range input
Mute toggle — one-click silence

Library & Browsing

Genre filter tags — All / Pop / Jazz / Classical / Electronic / Ambient
Live search — filters the track list as you type
Playlist sidebar — Liked Songs, Chill Vibes, Night Drive, Morning Jazz, Classics
Artist cards grid — responsive auto-fill layout with hover play button

Social / UX

Like system — heart any track; liked tracks appear in the Liked Songs playlist
Toast notifications — non-blocking feedback for actions (shuffle, repeat, like, playlist select)
Dynamic hero banner — updates to reflect the currently playing track
